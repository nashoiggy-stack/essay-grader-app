**Add — Scroll tracking:**
```typescript
import { useRef } from "react";
import { motion, useScroll, useTransform, useSpring } from "framer-motion";

const sectionRef = useRef<HTMLDivElement>(null);
const { scrollYProgress } = useScroll({
  target: sectionRef,
  offset: ["start start", "end end"],
});
const smoothProgress = useSpring(scrollYProgress, {
  stiffness: 100,
  damping: 30,
  restDelta: 0.001,
});
```

**Add — Animation phases using useTransform with single keyframe maps:**

Phase 1 — Hero fades (0 to 0.15):
```typescript
const heroOpacity = useTransform(smoothProgress, [0, 0.15], [1, 0]);
const heroScale = useTransform(smoothProgress, [0, 0.15], [1, 1.1]);
const heroBlur = useTransform(smoothProgress, [0, 0.15], [0, 15]);
const gridOpacity = useTransform(smoothProgress, [0, 0.12], [0.4, 0]);
```

Phase 2 — Card rises from below (0.05 to 0.3):
```typescript
const cardY = useTransform(
  smoothProgress,
  [0, 0.05, 0.3, 0.85, 1],
  ["110%", "110%", "0%", "0%", "-110%"]
);
```
This single keyframe map handles both rise (phase 2) and exit (phase 6). The card starts at 110% (off-screen below), rises to 0% by scroll 30%, stays until 85%, then exits to -110% (off-screen above).

Phase 3 — Card expands via scale (0.25 to 0.4):
```typescript
const cardScale = useTransform(
  smoothProgress,
  [0.05, 0.25, 0.4, 0.75, 0.85],
  [0.85, 0.85, 1, 1, 0.85]
);
const cardRadius = useTransform(
  smoothProgress,
  [0.25, 0.4, 0.75, 0.85],
  [40, 0, 0, 40]
);
```
The card is always 100vw x 100vh but scaled to 0.85 initially. It scales to 1 (appearing fullscreen) and then back to 0.85 before exit. Use `scale` NOT `width`/`height` for compositor performance.

**Apply motion styles — key rules:**
- Hero: `<motion.div style={{ opacity: heroOpacity, scale: heroScale, filter: heroBlurFilter }}>` where heroBlurFilter is derived via useTransform to produce `blur(Xpx)` string

- Feature cards: `<motion.div style={{ opacity: featureOpacity, y: featureY, scale: featureScale }}>` wrapping each Link
